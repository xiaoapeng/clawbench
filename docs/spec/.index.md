# ClawBench 系统设计规格

> 本目录包含 ClawBench 系统的设计规格文档，由定时任务每周自动更新。
> 文档目标是帮助读者（开发者或 AI）快速建立对系统的心智模型。

## 目录结构

- [总览](README.md) — 系统定位、模块地图、文档导航

### core/ — 核心业务

- [聊天流程](core/chat-flow.md) — 用户发消息到 AI 回复的完整链路，含 ACP 权限审批、多模式摘要
- [AI 后端抽象](core/ai-backend.md) — 双传输后端（CLI + ACP stdio）、BackendRegistry、流式解析
- [流式传输体系](core/streaming.md) — SSE 推送、重连、ACP 状态事件、HTTP 轮询降级
- [会话生命周期](core/session-lifecycle.md) — 创建、执行、取消、设置即时持久化、归档

### features/ — 功能特性

- [设置向导](features/setup-wizard.md) — 首次启动引导、供应商选择、API Key 加密
- [定时任务](features/scheduled-tasks.md) — cron 调度到 AI 执行到摘要推送，ACP 自动审批
- [语音合成](features/tts.md) — 引擎选择、文本清理、缓存策略
- [Web 终端](features/terminal.md) — 多标签 PTY 会话、虚拟键盘、键位配置、TUI 支持
- [Git 管理](features/git-management.md) — 历史浏览、Worktree、分支、标签
- [文件管理](features/file-management.md) — 浏览+覆盖层预览、符号提取、上传、归档
- [RAG 检索](features/rag.md) — 文档索引、向量化、搜索
- [推送通知](features/push-notifications.md) — JPush、权限待审推送、WebSocket 后备

### infra/ — 基础设施

- [认证与中间件](infra/auth-and-middleware.md) — 密码认证、localhost 旁路、请求链
- [SSH 隧道](infra/ssh-tunnel.md) — direct-tcpip、端口转发、EventBus 状态
- [Proxy 注册表](infra/proxy.md) — 端口管理、路由、前端展示
- [配置与自动发现](infra/config-and-discovery.md) — 零配置启动、DB-backed Agent、双传输、Model 发现
- [事件体系](infra/event-system.md) — EventBus、WebSocket、SSE、JPush、权限事件、缓冲重放

### client/ — 客户端

- [前端架构](client/frontend-architecture.md) — 单页布局、ACP 会话管理、composable 模式、store
- [Android 集成](client/android-integration.md) — JS Bridge、后台服务、SSH、推送策略

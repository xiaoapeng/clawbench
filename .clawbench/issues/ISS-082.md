## Description
VeCLI extractMetadata uses non-deterministic model selection from map iteration. When multiple model candidates are present in the metadata map, `range` over the map picks one arbitrarily. This means the selected model can vary between runs with the same input.

## Impact
Inconsistent model attribution in chat history. The displayed model name may not match the one actually used, causing user confusion and incorrect analytics.

## Suggestion
Use a deterministic selection strategy: sort the keys and pick the first, or use a priority list of known model identifiers. Ensure the same model is selected consistently for the same input.

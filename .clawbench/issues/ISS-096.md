## Description
ChatPanel.vue missing `show-thinking-detail` event handling — child components emit this event, but ChatPanel.vue does not listen for it or forward it up. The event is silently lost.

## Impact
Thinking detail toggle from child components has no effect. Users cannot expand/collapse thinking blocks through the intended UI pathway.

## Suggestion
Add an event handler for `show-thinking-detail` in ChatPanel.vue. Either handle the event locally (update the relevant state) or forward it to the parent component using `$emit`.

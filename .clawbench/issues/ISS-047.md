## Description
copyDir follows symlinks without checking that symlink targets stay under WatchDir. A symlink inside the source directory pointing to /etc/shadow or any file outside WatchDir will be silently copied, leaking data into the destination.

## Impact
Data exfiltration — arbitrary file read from the server filesystem. Any file readable by the ClawBench process can be copied into the project directory via a crafted symlink.

## Suggestion
Add symlink detection in copyDir — if entry.Type()&os.ModeSymlink != 0, resolve with filepath.EvalSymlinks and verify the target is under WatchDir, or skip symlinks entirely. This matches the protection in file_archive.go:107-116.

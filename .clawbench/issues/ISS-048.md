## Description
ServeFileDelete uses os.RemoveAll for directories, which follows symlinks inside the directory tree. If a directory under WatchDir contains a symlink pointing to /home/user, os.RemoveAll will follow it and delete files at the symlink target outside WatchDir.

## Impact
Arbitrary file deletion — symlinks within a project directory can cause deletion of files outside WatchDir, including user home directory contents.

## Suggestion
Walk the directory before deletion and verify no symlinks point outside WatchDir. Or walk and delete entries individually, resolving symlinks to check boundaries.

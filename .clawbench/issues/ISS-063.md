## Description
validateCreatePath — `req.Name` is not sanitized for path separators. A create request with `name: "subdir/../../etc/malicious"` can escape the intended directory.

## Impact
Arbitrary file or directory creation outside the project root. Could be used to plant malicious files, overwrite configurations, or create symlink targets.

## Suggestion
Strip path separators from `req.Name` or use `filepath.Base()` to extract only the final component. After joining with the parent path, verify the result is within the project root using a prefix check.

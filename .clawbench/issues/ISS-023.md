## Description
ChatPanelContent.vue is missing onExtractScheduledEvents callback in useChatStream setup. ChatPanel.vue (line 336) provides this callback, but ChatPanelContent.vue omits it. Scheduled task cards from SSE stream events will not be extracted/rendered in the tab-based content view.

## Impact
Silent data loss — users never see task creation cards when using ChatPanelContent (the tab-based layout).

## Suggestion
Add onExtractScheduledTasks: (msgs) => render.extractScheduledTasks(msgs) to the useChatStream config object in ChatPanelContent.vue.
